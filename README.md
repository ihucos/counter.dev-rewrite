

## Architecture goals
- Performance
- Development Speed

### Derived Philosophy

#### Dev/prod parity
The Production and Development environments is the same (as far as possible).

##### Couple Infrastrcuture as Code and Application Code
It's all just code.

##### Monorepo
Write the code to make it happen.

##### Vanilla is AI-Friendly
Leverage what the training data has seen, but also balance human ergonomics.

##### It will all run in one machine
No premature optimization: Scale vertically until horizontal scaling is needed.

##### Code capabilities, not features
Use AI, couple it to this project. In this case homegrown is better than Off-The-Shelf.
